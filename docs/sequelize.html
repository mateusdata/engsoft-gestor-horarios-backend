sequelize

Cria uma nova instância do Sequelize para a nossa aplicação.
Properties:
Name Type Description
process.env.DB_DATABASE string O nome do banco de dados.
process.env.DB_USERNAME string O nome de usuário para o banco de dados.
process.env.DB_PASSWORD string A senha para o banco de dados.
options object Um objeto contendo opções de configuração.
Properties
Name Type Description
host string O host do banco de dados.
port string A porta do banco de dados.
dialect string O dialeto do banco de dados a ser usado. Neste caso, 'mysql'.
dialectModule module:mysql2 O módulo de dialeto a ser usado. Neste caso, 'mysql2'.
